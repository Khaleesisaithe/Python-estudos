# 🐍 Estudos de Python

Bem-vindo ao meu repositório de estudos de **Python**.

Este espaço foi criado para registrar minha evolução na linguagem, desde os conceitos fundamentais até tópicos mais avançados. O objetivo é documentar meu aprendizado, organizar exercícios, desafios e projetos, além de servir como referência para futuras consultas.

## 🎯 Objetivos

* Aprender a sintaxe da linguagem Python.
* Desenvolver raciocínio lógico e resolução de problemas.
* Praticar algoritmos e estruturas de dados.
* Criar pequenos projetos para aplicar os conhecimentos adquiridos.
* Construir uma base sólida para Ciência de Dados, Automação, Desenvolvimento e Inteligência Artificial.
